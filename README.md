# ☕ Coffee Shop Sales Analytics Dashboard (Microsoft Excel)

An interactive **Coffee Shop Sales Analytics Dashboard** built entirely in **Microsoft Excel** to transform raw sales data into meaningful business insights. This project demonstrates how Excel can be used as a powerful Business Intelligence tool by combining Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and KPI cards to create a professional and user-friendly dashboard.

The dashboard enables users to monitor key performance indicators (KPIs), compare branch performance, analyze sales trends, and identify top-performing products through interactive visualizations.

---

## 📌 Project Overview

This dashboard provides a comprehensive analysis of coffee shop sales by tracking important business metrics, including:

- 💰 Total Revenue
- 📈 Total Profit
- 🧾 Total Transactions
- 🛒 Total Quantity Sold
- 🏪 Branch Performance
- 🌍 City-wise Sales Analysis
- ☕ Product Category Performance
- 📅 Monthly & Daily Sales Trends
- 📦 Top-Selling Products

Users can interact with the dashboard using slicers to filter data and explore insights from different perspectives.

---

## 📂 Dataset Information

The dashboard is developed using a **Coffee Shop Sales Dataset** containing **9,971 sales transactions** collected from multiple coffee shop branches.

### Dataset Features

| Column | Description |
|---------|-------------|
| Transaction ID | Unique identifier for each transaction |
| Date | Date of purchase |
| City | City where the sale occurred |
| Branch | Coffee shop branch |
| Shop Name | Store name |
| Product ID | Product identifier |
| Product Category | Product category |
| Unit Price | Selling price per item |
| Quantity Sold | Number of items sold |
| Revenue | Total revenue generated |
| Profit Margin | Profit earned from the sale |
